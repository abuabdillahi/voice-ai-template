## What to build

The second end-to-end tracer: a working voice conversation. After this slice, a signed-in user clicks the microphone, speaks, and hears the assistant reply in real time. The assistant has no tools, no memory, and no domain logic yet — just a system prompt instructing it to be a helpful conversational partner. This proves the full WebRTC + LiveKit Agents + OpenAI Realtime + frontend pipeline works end-to-end.

The realtime model is wired through a swappable factory so subsequent slices can change vendors with a config flip rather than a refactor.
