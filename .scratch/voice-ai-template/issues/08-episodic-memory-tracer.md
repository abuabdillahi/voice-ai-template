# Issue 08: Episodic memory tracer

Status: ready-for-agent
Category: enhancement

## Parent

`.scratch/voice-ai-template/PRD.md`

## What to build

Add fuzzy, similarity-searched memory for things that don't fit a named preference key — incidental facts mentioned in conversation ("I'm learning Spanish," "my daughter's name is Maya"). Backed by mem0 with its pgvector backend pointed at the same Supabase Postgres. The agent gets `remember(content)` and `recall(query)` tools. The sidebar gains a section listing recently-recalled facts.

After this slice, mentioning something incidental in one conversation, ending the session, starting a new one, and asking a related question produces an answer that references the earlier fact.
